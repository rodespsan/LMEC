Acta de Reunión


Revision de funcionalidad:


    Reparaciones:



  • Crear Reparaciones: 

    • Corregir la lista de trabajos  para que muestre el nombre y no la descripción.
    • Corregir la etiqueta de tipo de reparaciones a lista de trabajos.
    • Agregar titulo de trabajos realizados a la tabla de trabajos realizados.
    • Cambiar el titulo de la vista de Crear Reparaciones a Reparación.
    • Traducir  el mensaje de los campos obligatorios.
    • Quitar mensaje de error al crear una Reparación nueva.
    • Campo Descripción no es obligatorio.
    • Mostrar mensaje al crear reparación.
    • Si esta finalizado redireccionar a la vista ver sino te muestra el mensaje de que se ha guardado correctamente.


  • Listar Reparaciones:

    • ID no es necesario.
    • Orden debe mostrar el iD de la orden en formato folio.
    • Mostrar el número de trabajos realizados. (opcional)



  • Administración de Reparaciones:


    • Traducir el titulo, mensajes, menú lateral a español.
    • Quitar el ID de la tabla de administración de reparaciones.
    • Orden debe mostrar el iD de la orden en formato folio.
    • Mostrar el número de trabajos realizados. (opcional)
    • Corregir el botón de update.
    • Quitar del menú lateral la opción “create repair”.


  • Ver Reparaciones:


  • Quitar el ID de la tabla de administración de reparaciones.
  • Orden debe mostrar el iD de la orden en formato folio.
  • Listar los trabajos en viñetas.
  • Traducir el titulo, mensajes, menú lateral a español.
  • Quitar del menú lateral la opción “create repair”.




     Diagnóstico:


  • Crear Diagnóstico:

    • Solo mostrar los trabajos preventivos.
    • Mostrar mensaje al crear diagnóstico.
    • Etiqueta terminado cambiarlo a Finalizado.
    • Si esta finalizado redireccionar a la vista ver sino te muestra el mensaje de que se ha guardado correctamente.
    • Traducir menú lateral a español.
