Propuesta del proyecto

Resumen del Proyecto

Se realizará un sistema para el control de órdenes de servicio del laboratorio de mantenimiento y equipo de cómputo, que le de un mejor seguimiento a las órdenes de servicio, este sistema será web, en una arquitectura cliente servidor para que multiples usuarios tengan acceso al mismo tiempo con los diferentes roles de usuario.

Resumen de la Metodología

La metodología que se utilizará en el desarrollo de este proyecto se encuentra detallada en el Procedimiento de Diseño y desarrollo de aplicaciones Web del Sistema de Gestión de la Calidad de la Facultad de Matemáticas
