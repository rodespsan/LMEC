CP-01: Crear usuario

Propósito: Probar que los usuarios con rol de administrador pueden crear un usuario.
Prerequisitos:
Se debe haber iniciado sesión con un usuario con rol de administrador.
Debe existir al menos un rol activo.
Datos de Prueba:
  • Usuario = {válido, inválido, vacío}
  • Contraseña = {válido, inválido, vacío}
  • Confirmar contraseña = {válido, inválido, vacío}
  • Nombres = {válido, inválido, vacío}
  • Apellidos = {válido, inválido, vacío}
  • Correo electrónico = {válido, inválido, vacío}
  • Roles = {marcado(s), no marcado(s)}
  • Activo = {activo, inactivo}
Pasos:
  1. Ingresar al módulo de usuarios y seleccionar la opción crear usuario
  2. Introducir el usuario
  3. Introducir la contraseña
  4. Confirmar contraseña
  5. Introducir los nombres
  6. Introducir los apellidos
  7. Introducir el correo electrónico
  8. Marcar el(los) rol(es)
  9. Marcar o desmarcar activo
  10. Hacer clic en crear
  11. Verificar que se visualice el usuario que se acaba de crear y que los datos mostrados sean los mismos que se proporcionaron en el formulario.
Notas y Preguntas:
  • N/A
