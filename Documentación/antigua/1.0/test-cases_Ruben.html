TC-01: Crear cliente

Propósito: Probar que los usuarios con rol de administrador o recepcionista pueden crear un cliente.
Prerequisitos:
Se debe haber iniciado sesión con un usuario con rol de administrador o recepcionista.
Debe existir al menos un tipo de cliente activo.
Datos de Prueba:
  • Nombre de cliente = {válido, inválido, vacío}
  • Tipo de cliente = {seleccionado, no seleccionado}
  • Dependencia = {seleccionado, no seleccionado}
  • Dirección del cliente = {válido, inválido, vacío}
  • Nombre de contacto = {válido, inválido, vacío}
  • Correo electrónico = {válido, inválido, vacío}
  • Teléfono de oficina = {válido, inválido, vacío}
  • Teléfono celular = {válido, inválido, vacío}
  • Teléfono de casa = {válido, inválido, vacío}
  • Extensión de oficina = {válido, inválido, vacío}
  • Activo = {activo, inactivo}
Pasos:
  1. Ingresar al módulo de clientes y seleccionar la opción crear cliente
  2. Introducir el nombre de cliente
  3. Seleccionar el tipo de cliente
  4. Seleccionar la dependencia
  5. Introducir la dirección del cliente
  6. Introducir el nombre de contacto
  7. Introducir el correo electrónico
  8. Introducir el teléfono de oficina
  9. Introducir el teléfono celular
  10. Introducir el teléfono de casa
  11. Introducir la extensión de oficina
  12. Marcar o desmarcar activo
  13. Hacer clic en crear
  14. Verificar que se visualicé el cliente que se acaba de crear y que los datos mostrados sean los mismos que se proporcionaron en el formulario.

TC-05: Buscar clientes

Propósito: Probar que los usuarios con rol de administrador o recepcionista pueden buscar clientes con caracteristicas especificas.
Prerequisitos:
Se debe haber iniciado sesión con un usuario con rol de administrador o recepcionista.
Datos de Prueba:
  • Id = {vacío, no vacío}
  • Nombre = {vacío, no vacío}
  • Tipo de cliente = {vacío, no vacío}
  • Nombre de contacto = {vacío, no vacío}
  • Dependencia = {vacío, no vacío}
  • Activo = {si, no}
Pasos:
  1. Ingresar al módulo de clientes y seleccionar la opción administrar clientes.
  2. Introducir  el Id
  3. Introducir el nombre
  4. Introducir el tipo de cliente
  5. Introducir el nombre de contacto
  6. Introducir la dependencia
  7. Seleccionar una opción en el campo activo
  8. Presionar la tecla ENTER o hacer que el campo seleccionado pierda el foco.
  9. Verificar que en la lista de clientes desplegada los clientes fueron filtrados correctamete de acuerdo a los criterios de búsqueda.
Notas y Preguntas:
  • Si no coincide algún cliente con los criterios de búsqueda, el sistema despliega un mensaje indicando que no se encontraron resultados.
